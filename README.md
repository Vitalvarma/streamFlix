# 🎬 StreamFlix

A mini video streaming platform built with Next.js, PostgreSQL, Prisma, and Cloudinary.

## Features

- Video streaming
- Infinite scroll
- Search & category filtering
- Like system
- Dark mode
- Recommended videos

## Tech Stack

- Next.js
- Prisma ORM
- PostgreSQL
- React Query
- Cloudinary
- Tailwind CSS
